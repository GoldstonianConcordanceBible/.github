# Citation Format

All outputs should cite scripture and source layers in a stable format.

## Scripture Citation

Format:

`Book Chapter:Verse`

Examples:

- `Genesis 1:1`
- `John 1:1`
- `1 Enoch 1:1`
- `1 Meqabyan 1:1`

## Repository Citation

When citing internal GCB infrastructure, use:

- repository name
- file name
- stable section or row when possible

Example:

`canonical-index / canon/manifest.csv`
`gcb-scripture-dataset / data/verses.csv`
`gcb-theological-entities / data/theological_entities.csv`

## Priority Rule

Cite in this order:

1. scripture reference
2. canonical source
3. ontology source
4. agent synthesis
